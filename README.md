## Описание уязвимости: CVE-2024-57279

### Краткое описание
CVE-2024-57279 — это Reflected Cross-Site Scripting (XSS) уязвимость, найденная в приложении LDAP User Manager версии ce92321 и более ранних. Она позволяет атакующему внедрить и выполнить вредоносный JavaScript-код в браузере ничего не подозревающего пользователя.

### Детали
*   **Тип уязвимости:** Reflected XSS (CWE-79)
*   **Затронутый компонент:** Файл `/setup/index.php`
*   **Уязвимый параметр:** `returnto`
*   **Вектор атаки (AV):** Сетевой (Network)
*   **Сложность атаки (AC):** Низкая (Low)
*   **Причина:** Недостаточная санитизация (очистка) пользовательского ввода.
*   **Оценка CVSS v3.1:** 5.4 (MEDIUM) — `CVSS:3.1/AV:N/AC:L/PR:L/UI:R/S:C/C:L/I:L/A:N`
*   **Дата публикации:** 7 февраля 2025

### Потенциальный ущерб
Успешная эксплуатация этой уязвимости позволяет злоумышленнику:
*   Украсть cookies и сессионные токены жертвы.
*   Подделать содержимое веб-страницы.
*   Выполнить действия от имени пользователя на сайте.
*   Перенаправить пользователя на фишинговый сайт.

## PoC-скрипт (Proof of Concept)
В файле `exploit_demo.py` представлен демонстрационный скрипт, который имитирует принцип работы эксплойта для данной уязвимости. Он формирует вредоносный URL-запрос и показывает, как внедренный код отражается в ответе сервера.

### Использование
1.  Убедитесь, что у вас установлен Python 3.
2.  Установите необходимую библиотеку:
    ```bash
    pip install requests
    ```
3.  Запустите скрипт в демо-режиме (без атаки на реальную цель):
    ```bash
    python exploit_demo.py
    ```
4.  Для проверки на своей тестовой системе (только с разрешения!):
    ```bash
    python exploit_demo.py http://адрес-вашего-сайта/setup/index.php
    ```
